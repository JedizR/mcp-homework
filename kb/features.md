# Features

OmniContext is a memory layer that runs alongside Claude. Claude doesn't retain context between sessions — this stores it locally so you don't have to re-explain yourself every time.

- Stores named memories with a timestamp and optional tags
- Retrieves memories by keyword or tag
- Updates or deletes a stored memory by name
- Runs entirely local — no cloud sync, nothing sent anywhere
- Exports the full memory store as plain JSON
