# Competitive analysis

| Tool | Strengths | Weaknesses |
|------|-----------|------------|
| OmniContext | Fully local, no setup beyond running the server, plain JSON storage you can read and edit by hand | No semantic search, manual tagging required, no web UI |
| Claude built-in memory | Zero setup, works automatically, Anthropic manages it | Only available on Claude.ai paid plans, no export, you can't see what's stored |
| mem0 | Semantic search, cloud sync, works across tools | Requires an account, data leaves your machine, adds a network dependency |
| Plain notes (Obsidian, etc.) | Total control, portable, works everywhere | Not connected to Claude at all — you have to paste context in yourself |
